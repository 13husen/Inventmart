## Inventmart

<p>
Simple Desktop application for Inventory Management with Material Design, builded with Spring Boot, Hibernate and JavaFX.
</p>


### User Default: <br/>

<p> 
  <b>Email</b>: admin@admin.com<br/>
  <b>Password</b>: 123<br/>
  <b>Role</b>: Admin
</p>
